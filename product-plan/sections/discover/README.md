# Discover

The YouTube channel hub for finding cooking creators, following them, and saving recipes from their videos.

## Components

- `Discover.tsx` — Main view with feed/channels toggle
- `VideoCard.tsx` — Video card for feed with save button
- `ChannelCard.tsx` — Channel card with follow button
- `ChannelDetail.tsx` — Full channel page with videos
- `CategoryChip.tsx` — Category browsing chips
- `RecipePreviewModal.tsx` — AI recipe extraction preview

## Key Features

- Browse video feed from followed channels
- Search and discover YouTube cooking channels
- Follow/unfollow channels
- Preview AI-extracted recipe before saving
- Category-based channel browsing
