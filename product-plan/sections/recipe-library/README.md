# Recipe Library

The central hub where users save, browse, and manage all their recipes.

## Components

- `RecipeList.tsx` — Main list view with grid/list toggle, search, filters
- `RecipeCard.tsx` — Individual recipe card for grid/list display
- `RecipeDetail.tsx` — Full recipe detail view with scaling, actions
- `AddRecipeMenu.tsx` — Floating action button menu for adding recipes
- `ScanSession.tsx` — Camera scanning workflow for cookbooks

## Key Features

- Save recipes from YouTube, websites, scanned cookbooks, or manual entry
- Browse in grid or list view with search and source filtering
- View detailed recipe info with ingredients, instructions, nutrition
- Scale servings with auto-calculated ingredient quantities
- Add to shopping list or schedule in meal plan
