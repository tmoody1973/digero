# Recipe List and Detail Views

## Feature Description

Display user's saved recipes in a scrollable list with search/filter. Tapping a recipe opens a detail view showing full ingredients and instructions in a cook-friendly format.

## Source

Product Roadmap - Week 1: Foundation and Core Flow (Feature #4)

## Effort Estimate

Medium (M) - 1 week

## Priority

High - Essential for recipe browsing and viewing
